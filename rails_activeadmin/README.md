# Rails ActiveAdmin

## Starting

### Setup

```sh
$ bundle install
$ bundle exec rake db:setup
```

### Run

```sh
$ bundle exec rails s
$ open http://localhost:3000/admin
```

You enter the credential listed below:

* **Email**: `admin@example.com`
* **Password**: `password`
